Residential Controller

The program to develop is a controller set up in:

  • 10 storey building
  • 2 elevator cages

This controller is capable of supporting two main events:

  1. George presses a call button to request an elevator, the main controller selects an available cage and it is sended to the person based on two parameters provided by the input " press button"
    • The floor where the person is (0 to 10)
    • The direction in wich he wants to go (up or down)
  2. George enters an elevator, press to which floor he wanna go, and the elevator moves to the requested floor. The inputs are:
    • The elevator used
    • The requested floor

The developed controller must be provided in a single file and must include both methods:

  1. Method 1:
    • INPUT: RequestElevator (The floor where George is (requestedFloor), direction to go (Up/Down))
    • RETURN: The choosen elevator an move it (to the floor where George makes the call)
  2. Method 2:
    • INPUT: RequestFloor (Elevator (The elevator choosen), RequestedFloor (Which floor George wanna go))
    • RETURN: Move the elevator to the floor that George request
